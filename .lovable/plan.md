

## Faisal Ansari — Data Analyst Portfolio Website

### Design Direction
Inspired by the Payscale brand imagery: **dark teal/green gradient background**, **neon green accents**, **purple highlights**, clean modern typography (Poppins for headings, Inter for body). Smooth animations, glass-morphism cards, and gradient overlays for a premium feel.

### Sections (Single-page scrolling layout)

1. **Hero Section**
   - Name: "Faisal Ansari" with animated gradient text
   - Title: "Data Analyst"
   - Tagline: "Transforming data into actionable insights"
   - CTA buttons: "View Projects" and "Download Resume"
   - Subtle particle/grid background animation

2. **About Me**
   - Brief bio from resume
   - Highlight: data visualization, dashboard development, business insights
   - Contact info: email, phone, LinkedIn, location

3. **Skills**
   - Animated skill cards/badges with icons
   - Categories: Excel (Pivot Tables, Power Query), SQL (Joins, Aggregations), Power BI / Tableau, Python (Pandas, Matplotlib), Data Cleaning & Visualization

4. **Projects**
   - Card grid with hover effects
   - **Car Sales Dashboard** – Power BI (KPIs, DAX measures, revenue tracking)
   - **Hospital Emergency Room Data** – Excel & SQL (pivot reports, customer behaviour)
   - **Python Data Analysis Project** – Pandas & Matplotlib (trends, reports)
   - Each card shows: title, tools used, description, link button

5. **Education**
   - Timeline layout
   - 10th & 12th Science – Radcliffe School (2022-2024)
   - B.Com – Mumbai University (2024-Present, 2nd Year, ML & AI/DS focus)

6. **Certifications**
   - Badge-style cards for: Excel, Power BI, Python, SQL

7. **Contact**
   - Contact form (name, email, message)
   - Social links: LinkedIn, Email
   - Location info

### Navigation
- Sticky top navbar with smooth scroll to sections
- Mobile hamburger menu

### Features
- Fully responsive
- Smooth scroll navigation
- Resume download button (placeholder PDF link)
- Animated section reveals on scroll
- Dark theme with teal/green/purple accent palette
